# Basic Network Sniffer

## Project Overview

This project was developed as part of the **CodeAlpha Cyber Security Internship**.

The Basic Network Sniffer is built using **Python** and the **Scapy** library. It captures live network packets and displays important packet information such as the source IP address, destination IP address, protocol, packet length, timestamp, and packet summary.

This project helps beginners understand how network packets travel across a network and introduces the fundamentals of packet analysis.

---

## Features

- Capture live network packets
- Display Source IP Address
- Display Destination IP Address
- Identify Protocol (TCP / UDP)
- Display Packet Length
- Display Timestamp
- Display Packet Summary

## How to Run

1. Install Python.
2. Install Npcap.
3. Install the required library:

```bash
pip install -r requirements.txt
```

4. Run the program:

```bash
python sniffer.py
```
